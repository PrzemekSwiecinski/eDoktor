# eDoktor

(PL) Aplikacja do umawiania wizyt ze spacjalistami online. Napisana w języku React/TypeScript (Frontend) oraz PHP (backend) wraz z relacyjną bazą danych SQL. Aplikacja posiada: 
- logowanie użytkownika, lekarza i admina;
- możliwość umawiania wizyt na określoną godzinę;
- panel admina z możliwością dodawania i usuwania lekarza;
- panel użytkownika z wizytami oraz danymi osobowymi;
- panel lekarza z wizytami;
- możliwość oceniania lekarzy;
- kilka innych mniej lub bardziej użytecznych podstron (łącznie 14 widoków).

(EN) An application for scheduling appointments with specialists online. Written in React/TypeScript (frontend) and PHP (backend) with a relational SQL database, the application features:
- login for users, doctors, and admins;
- the ability to schedule appointments for specific times;
- an admin panel with the ability to add and remove doctors;
- a user panel with appointments and personal details;
- a doctor panel with appointments;
- the ability to rate doctors;
- several other subpages (14 views in total).
